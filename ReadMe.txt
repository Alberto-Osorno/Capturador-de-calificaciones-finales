CAPTURADOR DE CALIFICACIONES
----------------------------
Este programa permite capturar calificaciones de estudiantes y gestionarlas de forma sencilla.


REQUISITOS
----------
Antes de poder compilar el programa es necesario tener instalado el JDK 17 o una versión más actual del mismo. Tambien es necesario haber descargado el archivo "ADA2_U1_T1.zip" y haber extraido su contido ("Manipulador.java" y "Main.java") en una carpeta propia.


PASOS PARA COMPILAR
-------------------
Una vez con realizadas esas acciones abre la terminal de tu computadora y navega hasta la carpeta en la que se extrajo el contenido de "ADA2_U1_T1.zip". Una vez ahí ejecuta el comando:

    javac -d ./bin *.java

Este comando compila todos los archivos ".java" de la carpeta y guarda todos los archivos ".class" resultantes en una nueva carpeta llamada "bin". 


PASOS PARA EJECUTAR
-------------------
Navega hasta la carpeta bin que se acaba de crear y se ejecuta el comando:

    java Main

Este comando hará que se empiece a ejecutar el programa para capturar calificaciones.
